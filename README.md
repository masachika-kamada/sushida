# 寿司打自動化

タイピングゲーム「[寿司打](http://typingx0.net/sushida/)」をPythonで自動化した

<https://user-images.githubusercontent.com/63488322/165005032-7f74ac42-374f-4b46-a076-05b66a1ab5fe.mp4>

# 使い方

(動作確認環境：windows 10)
1) リポジトリのクローン
2) chromeのバージョンに対応したchromedriverをインストール
3) 実行

```bash
python main.py
```
